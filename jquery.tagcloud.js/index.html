jquery.tagcloud.js

This is a forked version of addywaddy's jquery.tagcloud.js plugin. This version adds the support for JSON data as a source instead of only relying on existing tags.

Usage without JSON data source. 

	$(document).ready(function(){
	   $("#tagcloud").tagcloud();
	});
